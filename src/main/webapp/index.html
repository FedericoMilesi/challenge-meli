Desafío Mercado Libre: Operación Fuego de Quasar

Sobre

Este es un programa desarrollado para una prueba técnica de Mercado Libre. El programa consiste en interceptar la posición y el mensaje de una nave en el espacio mediante satélites. Los satélites son tres y sabemos las coordenadas de cada uno (todo sucede en un plano cartesiano), por nuestra parte debemos ingresar las distancias estimadas entre los satélites y la nave emisora del mensaje, al igual que el mensaje recibido por cada satélite (que es un fragmento del mensaje original). De este modo podremos determinar tanto la ubicación como el mensaje completo de la nave.

Como usar

Debemos utilizar la herramienta web Postman y generar una petición HTTP de tipo POST a la dirección challengeMELI/api/topsecret con la siguiente información en el cuerpo de la petición:


[

       {

            "nombre": "kenobi",

            "distanciaAlEmisor": 0.0,

            "mensajeRecibido": ["", "", "", "", ""]

       },

       {

            "nombre": "skywalker",

            "distanciaAlEmisor": 0.0,

            "mensajeRecibido": ["", "", "", "", ""]

       },

       {

            "nombre": "sato",

            "distanciaAlEmisor": 0.0,

            "mensajeRecibido": ["", "", "", "", ""]

       }

]

Sin modificar el formato, cambiamos la distancia al emisor estimada de cada satélite por otro número real, y dentro de las comillas en negrita ingresamos cada palabra del mensaje que recibieron como a continuación:


[

       {

            "nombre": "kenobi",

            "distanciaAlEmisor": 663.4,

            "mensajeRecibido": ["este", "", "", "mensaje", ""]

       },

       {

            "nombre": "skywalker",

            "distanciaAlEmisor": 264.7,

            "mensajeRecibido": ["", "es", "", "", "secreto"]

       },

       {

            "nombre": "sato",

            "distanciaAlEmisor": 447.7,

            "mensajeRecibido": ["este", "", "un", "", ""]

       }

]

Luego de ingresar esta información obtendremos (si es posible) la localización de la nave y el mensaje esperado.

También es posible ingresar los datos para cada satélite de forma individual. Para ello, debemos generar una petición POST a la dirección challengeMELI/api/topsecret_split/nombredelsatelite con el siguiente formato:


{

       "distanciaAlEmisor": 0.0,

       "mensajeRecibido": ["", "", "", "", ""]

}

Herramientas utilizadas


Apache NetBeans IDE 12.6 (entorno de desarrollo)

Java Development Kit 8 (para programar en Java)

Java EE Web Api 7.0 (para crear la aplicación Web)

Postman API Platform (para generar las peticiones HTTP en formato JSON)

Oracle Glassfish Server (para probar el programa fuera de línea)

Geogebra Calculator Suite (para poder verificar los resultados del algoritmo matemático del programa)
